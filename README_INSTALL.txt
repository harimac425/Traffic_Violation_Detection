# Traffic Violation Detection System - Installation Guide

### ⚠️ IMPORTANT: Python Version Requirement
This application strictly requires **Python 3.10**.
Do NOT use other versions (like 3.11 or 3.14), as the AI models are verified only for 3.10.

### 🛠️ One-Click Fix for Installation Errors:
If you are seeing "ERROR" or "AttributeError":

1.  **Run the Installer**: Double-click **`install_python_310.bat`** in this folder.
2.  **Wait**: It will automatically download and install the verified Python 3.10 environment.
3.  **Run the App**: Re-launch `TVDS.exe`. It will now find the correct version and work perfectly.

### GPU Support (NVIDIA)
If you have an NVIDIA GPU, ensure you have **CUDA 11.8** or **12.1** installed for maximum performance.
